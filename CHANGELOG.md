# Changelog

All notable changes to this project will be documented in this file.

---

## 1.1.0

🎉 Added Flutter Form Validators and customizable validation messages.

### ✨ Added

* 📝 Form Validators for all built-in validators.
* 🎨 Custom validation messages using the `messages` parameter.
* 🏗️ BaseFormValidator for reusable form validation.
* 📖 Flutter Form validation examples.
* 🧪 Unit tests for all Form Validators.

### 🚀 Improved

* Improved package architecture.
* Improved Flutter integration.
* Improved documentation.
* Improved example application.
* Expanded test coverage.
